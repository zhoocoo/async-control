## AsyncControl

一个使用promise来解决并发异步的方案

### 功能

- [x] 并发的基础控制
- [x] 并发持续发生时，向并发池子中继续添加并发请求
- [x] 每次push添加新异步集合时，正确监听回调
- [ ] 重试机制
- [ ] 跳过机制
- [ ] 暂停机制
